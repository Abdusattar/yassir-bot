## Архитектура

```
bot.py              ← long-polling
config.py           ← BOT_PROFILE=male, SUPER_ADMIN_IDS
core/
  handlers.py       ← process_message()
  scheduler.py      ← расписание
  ai.py             ← промпты: _TADABBUR_POST_SYSTEM, _NASIHA_SYSTEM
  sampler.py        ← sample_ayah/hadith (motiv_score≥3)
  transfers.py      ← авто-переводы
  i18n.py           ← T(key,lang)
  content.py        ← PROJECT_INFO
sources/hadiths.db  ← аяты + хадисы
```
