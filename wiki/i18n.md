# Многоязычность (i18n)

## Принцип

Язык группы хранится в `groups.lang` (ru/ky/uz/kk/en).  
Читается через `get_group_lang(group_dict)` из `core/db.py`.  
Все пользовательские тексты идут через `T(key, lang, **kwargs)` из `core/i18n.py`.

## Функции i18n.py

| Функция | Назначение |
|---|---|
| `T(key, lang, **kwargs)` | Перевод строки по ключу. Fallback: ru |
| `help_student(group_tasks, gtype, lang)` | Справка студента (с учётом типа группы) |
| `help_admin(lang)` | Справка устаза |
| `task_name(key, lang)` | Название задания на языке группы |
| `lang_instruction(lang)` | Инструкция для AI модели (на английском) |

## lang_instruction

**Обязательно на английском**, чтобы модель не путалась с языком вывода:

```python
"IMPORTANT: write the ENTIRE response in Kyrgyz language (code: ky). Do not mix languages."
```

Если промпт на русском + инструкция "пиши по-кыргызски" → модель путается, пишет минимум.  
Если промпт и инструкция на английском → модель правильно переключается на нужный язык.

## Ключи переводов (основные)

- `ask_name`, `welcome`, `report_accepted`, `report_already`, `done_all`
- `tasks_left`, `streak_praise`, `no_active_group`
- `prep_congrats`, `prep_failed`, `prep_reminder_active`, `prep_reminder_inactive`
- `prep_link_ru`, `prep_link_ky`, `prep_group_link`, `prep_no_group`

## Задания (task_name)

```python
TASK_NAMES_I18N = {
    "m": {"ru": "📿 Мантия (намаз)", "ky": "📿 Намаз", ...},
    "r": {"ru": "📖 Заучивание (40+40)", "ky": "📖 Жаттоо (40+40)", ...},
    "t": {"ru": "📝 Муфрадат", "ky": "📝 Муфрадат", ...},
    "j": {"ru": "🎵 Таджвид", "ky": "🎵 Тажвид", ...},
    "n": {"ru": "📚 Нахв", "ky": "📚 Нахв", ...},
    "h": {"ru": "🕌 Хадис", "ky": "🕌 Хадис", ...},
}
```

## Добавить новый язык

1. Добавить ключ в `LANG_NAMES` (значение — английское название: "Arabic")
2. Добавить переводы во все словари `_STRINGS`, `_HELP_STUDENT`, `_HELP_ADMIN`, `TASK_NAMES_I18N`
3. Язык группы установить через `/setlang XX` в чате
